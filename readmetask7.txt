# Task 7 - Identify and Remove Suspicious Browser Extensions

## Objective
To identify and remove potentially harmful browser extensions.

## Browser Used
Google Chrome

## Extensions Reviewed
1. Google Docs Offline (Safe - Official Google Extension)
2. WPS PDF - Read, Edit, Fill, Convert (Third-party)

## Suspicious Findings
The WPS PDF extension:
- Had permission to read and change data on websites
- Was added by a third-party developer
- Was not frequently used

## Action Taken
The WPS PDF extension was removed to improve browser security.

- Learned about browser extension permissions
